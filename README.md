# Zomato Restaurant insight dashboard
## This project presents an interactive Tableau dashboard analyzing restaurant data from Zomato to uncover insights into sales performance, customer preferences, and geographic trends.The dashboard enables users to explore key metrics such as total restaurants, cuisines, sales, ratings, and city-wise distribution, helping stakeholders make data-driven decisions in the food and restaurant industry.
## Dataset Used

 # Key features
- Visualized sales trends over time using line/area charts
- Compared cuisine performance based on sales and ratings
- Displayed city-wise restaurant distribution using treemap visualization
- Highlighted Top 5 restaurants based on sales and rating metrics

  # Project Insight
- New Delhi has the highest concentration of restaurants
- North Indian and Chinese cuisines dominate sales and popularity
- A small percentage of restaurants generate a major share of revenue
- Sales trends fluctuate across years, indicating demand variation
- Average ratings indicate moderate customer satisfaction levels
  # Final conclusion
This project demonstrates how raw restaurant data can be effectively transformed into meaningful business insights using **Tableau**. The dashboard highlights important patterns in sales performance, customer preferences, and geographic distribution, helping to identify high-performing cuisines and locations such as New Delhi. Overall, it showcases strong data visualization, analytical thinking, and the ability to support data-driven decision-making through an interactive and user-friendly dashboard.
